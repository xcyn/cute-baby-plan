## 一、项目介绍
- 萌宝计划-活动页
- 项目目的：用于前端技术分享，包括前后端端开发部署。
## 二、功能介绍
- 简单的一个活动项目，主要包含微信分享、 用户系统、参赛功能和投票功能及数据统计。
- 前端技术栈：vue、 vue-cli
- 后端技术栈：node、 koa、 mongoose
- web服务器: nginx
- 部署类: jenkins

- jenkins部署总结: <a href="https://www.jianshu.com/p/9049efacdbdb">地址</a>
